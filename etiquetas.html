EMBED

MARK

En este párrafo estamos usando la etiqueta mark.

La etiqueta mark se usa para destacar una parte de texto que es relevante para el usuario en ese momento.

Se suele usar en los resultados de búsquedas, en este caso yo lo he usado con el nombre de la etiqueta, ya que es destacable en esta sección por ser importante para el usuario.

TIME
